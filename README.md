### Problema
#### Una tienda ofrece diferentes tipos de cafe,
- Espresso
- Latte
- Cappuccino
#### Pero donde los clientes pueden personalizar su pedido agregando diferentes toppings como:
- Leche
- Chocolate
- Canela
#### Se requiere un sistema que permita añadir estos toppings de forma que no se deba modificar las clases de cada tipo de cafe.

### Patrón utilizado
Decorator
#### Utilice este patron porque permite agregar diferentes tipos de ingredientes extra, cuantos quiera.
### Estructura del proyecto
- Cafe
- Espresso
- Latte
- Cappuccino
- DecoratorCafe
- Leche
- Chocolate
- Canela
